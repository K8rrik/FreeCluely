# FreeCluely

FreeCluely — это скрытый ассистент для macOS, созданный для незаметного взаимодействия с Google Gemini AI. Приложение работает в виде минималистичного оверлея, который не привлекает внимания и позволяет получать помощь ИИ, анализируя содержимое вашего экрана в реальном времени.

## Возможности

- **Скрытый оверлей**: Минималистичный дизайн, который не мешает работе.
- **Анализ экрана**: Анализ содержимого экрана для получения релевантной помощи ИИ.
- **Голосовой режим**: Транскрипция речи в текст в реальном времени с использованием Deepgram (модель Nova-2).
- **Умные подсказки**: Проактивные подсказки ИИ на основе контекста разговора.
- **Окно транскрипции**: Отдельное окно в стиле терминала для логов транскрипции.
- **Глобальная горячая клавиша**: Быстрый доступ к ассистенту из любого места.

## Установка и Запуск

### 1. Получите API ключи
1. **Google Gemini**: Посетите [Google AI Studio](https://aistudio.google.com/) и создайте API ключ.
2. **Deepgram**: Посетите [Deepgram Console](https://console.deepgram.com/) и создайте API ключ (требуется для голосового режима).

### 2. Настройте приложение
1. В корневой папке проекта (рядом с этим файлом) создайте файл с названием `.env`.
2. Вставьте в него ваши ключи:
   ```bash
   GEMINI_API_KEY=ваш_api_ключ_здесь
   GEMINI_MODEL=gemini-3-pro-preview
   GEMINI_FAST_MODEL=gemini-2.5-flash // Быстрая модель для ответов во время диалога
   DEEPGRAM_API_KEY=ваш_deepgram_ключ_здесь
   ```

### 3. Соберите приложение
Откройте терминал в папке проекта и запустите команду сборки:
```bash
./build_app.sh
```
Этот скрипт автоматически скачает необходимые зависимости и создаст готовое приложение.

### 4. Запустите
После успешной сборки в папке `Build` появится `FreeCluely.app`. Запустите его.

## Горячие клавиши

- **Cmd + /**: Глобальная горячая клавиша для показа окна и фокуса на вводе.
- **Cmd + Shift + A**: Анализировать содержимое экрана и получить помощь ИИ.
- **Cmd + Shift + W**: Показать/скрыть окно приложения.
- **Cmd + Shift + C**: Очистить текущую сессию и начать новую.
- **Cmd + Shift + H**: Показать/скрыть историю.
- **Cmd + Стрелки**: Перемещение окна приложения.

## Устранение неполадок
- **Отказано в доступе**: Приложению требуются разрешения на запись экрана для анализа контента и захвата системного аудио.
  Перейдите в **Системные настройки > Конфиденциальность и безопасность > Запись экрана** и разрешите доступ для FreeCluely.
- **Проблемы с голосовым режимом**: Голосовой режим требует macOS 13.0 или новее. Убедитесь, что ваш API ключ Deepgram правильно указан в `.env`.
